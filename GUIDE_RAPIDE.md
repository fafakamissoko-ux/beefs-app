## 📱 Les 5 Onglets

### 1. 💬 CHAT
- Messages en direct
- **8 réactions rapides** en bas
- +1 point par réaction

### 2. 👥 QUEUE
- File d'attente des challengers
- Rejoindre la queue
- Voir sa position

### 3. 🤖 AI CHECK
- Vérification des faits par IA
- **Système de gifts** (4 types)
- Coûte 10-100 points

### 4. 📊 STATS
- **Leaderboard** complet
- Top débatteurs
- Séries de victoires 🔥

### 5. 🎯 ENGAGE
- **Sondages** en direct (+5 pts)
- **Prédictions** (parier des points)
- Gain potentiel affiché

---

## 💰 Système de Points

### Gagner des Points :
- ⏱️ **10 pts/min** : Regarder le stream (automatique)
- 🎭 **+1 pt** : Envoyer une réaction
- 📊 **+5 pts** : Voter dans un sondage
- 💬 **+2 pts** : Envoyer un message
- 🏆 **+50 pts** : Gagner un débat

### Dépenser des Points :
- 🎁 **10-100 pts** : Envoyer des gifts
- 🎯 **Variable** : Faire des prédictions

---

## 🎬 Créer un Clip

1. Cliquez sur **"CRÉER CLIP (30s)"** en haut à droite
2. Le clip des 30 dernières secondes est créé
3. Cliquez sur 📤 pour partager

---

## 🔥 Mode Chaos

Quand la **tension atteint 100%** :

- 🌊 Effets visuels spectaculaires
- ⚡ Particules animées
- 🔴 Vignette rouge
- 🎆 Alerte "MODE CHAOS ACTIVÉ"

**Pour activer** : Spammez le bouton **[TAP!]** en bas

---

## 🎯 Faire une Prédiction

1. Allez dans l'onglet **🎯 ENGAGE**
2. Choisissez votre option (Host ou Challenger)
3. Sélectionnez le montant (slider ou boutons rapides)
4. Cliquez sur **"PARIER X POINTS"**
5. Gagnez jusqu'à **2.5x vos points** !

---

## 📊 Voir le Leaderboard

1. Cliquez sur l'onglet **📊 STATS**
2. Basculez entre **POINTS** et **VICTOIRES**
3. Les 3 premiers ont des icônes spéciales :
   - 👑 1er : Couronne or
   - 🥈 2e : Trophée argent
   - 🥉 3e : Trophée bronze

---

## 🎁 Envoyer un Gift

1. Allez dans l'onglet **🤖 AI CHECK**
2. Descendez jusqu'à "ENVOYER UN GIFT"
3. Cliquez sur **"Offrir Gift"**
4. Choisissez parmi :
   - 🔥 **Flamme** (10 pts)
   - ⚡ **Éclair** (25 pts)
   - 👑 **Couronne** (50 pts)
   - 💎 **Diamant** (100 pts)

Le gift apparaîtra en animation sur l'écran !

---

## 🎭 Envoyer des Réactions

1. Onglet **💬 CHAT**
2. Scrollez en bas
3. Section "RÉACTIONS RAPIDES"
4. Cliquez sur : 🔥 👏 😂 😱 💀 💯 👀 🎯

L'emoji volera sur l'écran principal !

## 🏆 Stratégie pour Maximiser vos Points

### Niveau Débutant (0-1000 pts)
1. ⏱️ Restez connecté (10 pts/min gratuit)
2. 🎭 Envoyez des réactions régulièrement
3. 📊 Votez dans tous les sondages

### Niveau Intermédiaire (1000-5000 pts)
1. 🎯 Faites des prédictions prudentes (paris faibles)
2. 💬 Participez au chat
3. 🎁 Envoyez des petits gifts (flammes)

### Niveau Avancé (5000+ pts)
1. 🎯 Faites de grosses prédictions
2. 🎁 Envoyez des gifts premium (diamants)
3. 🏆 Montez dans le leaderboard

## 🐛 Dépannage

### Le serveur ne démarre pas
```bash
npm install
npm run dev
```

### Les composants ne s'affichent pas
- Rafraîchissez la page (Ctrl+R)
- Videz le cache (Ctrl+Shift+R)

### Les points ne s'incrémentent pas
- C'est normal, le gain passif prend 1 minute
- Interagissez pour gagner des points immédiatement
